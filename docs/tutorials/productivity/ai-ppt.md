# 用AI一键生成PPT大纲+内容

## 📌 一句话总结
用 ChatGPT / Kimi 生成 PPT 大纲和正文，再用 Gamma / Beautiful.ai 一键渲染成可演示的 PPT。

## 🛠️ 用到的工具
- **ChatGPT** 或 **Kimi**（生成大纲和正文）
- **Gamma.app**（一键渲染成 PPT，免费版够用）
- 可选：**Beautiful.ai** / **SlidesAI**（替代方案）

## ⏱️ 耗时 & 难度
- 预计时间：15分钟
- 难度：⭐ 新手友好

## 🎯 最终效果
一份结构完整、排版专业的 PPT 文件（.pptx），包含：
- 封面页
- 目录页
- 内容页（3-8页，视主题而定）
- 总结页

## 📝 操作步骤

### Step 1：用AI生成PPT大纲

打开 ChatGPT 或 Kimi，输入以下 Prompt（以"季度工作总结"为例）：

```
你是一位资深产品经理，请帮我生成一份季度工作总结 PPT 的大纲和每页正文。

要求：
1. 共 6-8 页
2. 每页包含：标题 + 3-5 个要点（bullet points）
3. 语言简洁专业，适合向管理层汇报
4. 输出格式：页码 | 标题 | 正文要点

主题：Q2 产品运营季度总结
背景：我们是一款 AI 工具导航产品，Q2 完成了 v2.0 上线、用户增长 40%、新增 3 个核心功能
```

ChatGPT 会生成类似这样的输出：

```
第1页 | 封面
- Q2 产品运营季度总结
- AI工具导航 v2.0
- 汇报人：XXX

第2页 | 目录
- 核心数据概览
- 重点功能上线
- 用户增长情况
- 竞品对比
- Q3 规划

第3页 | 核心数据概览
- DAU 突破 50万，环比增长 40%
- 用户留存率 65%（行业平均 45%）
- NPS 评分 72 分，同比提升 15 分
...
```

### Step 2：用 Gamma 一键渲染成PPT

1. 打开 https://gamma.app（免费注册）
2. 点击 **"Create with AI"**
3. 选择 **"Paste in text"**
4. 把 Step 1 生成的大纲**全部粘贴**进去
5. 点 **"Generate"**，等待 30 秒
6. Gamma 会自动生成排版、配色、图标

**生成完后：**
- 可以直接在线演示
- 点 **"Export" → "PowerPoint"** 下载 .pptx 文件
- 下载后可以用 PowerPoint / WPS 继续微调

### Step 3（可选）：用AI优化文案

如果觉得 Gamma 生成的文案不够好，可以把 PPT 的正文复制回 ChatGPT，让它润色：

```
请帮我润色以下 PPT 文案，要求：
1. 每页不超过 30 字
2. 语言更有冲击力
3. 适合演示场景（不是读稿）
```

## ⚠️ 注意事项 / 避坑

1. **Gamma 免费版限制**：每月可生成 10 个 PPT，够个人使用；商用建议付费
2. **AI 生成的数据要核实**：ChatGPT 可能会"编造"数据，务必替换成真实数字
3. **版权问题**：Gamma 的图标/图片是免版税的，可以直接用；如果要用网络图片，注意版权
4. **不要完全依赖AI**：AI 生成的是"草稿"，一定要人工调整结构和重点
