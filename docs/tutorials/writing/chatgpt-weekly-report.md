# 用ChatGPT 10分钟写完周报

## 📌 一句话总结
用 ChatGPT 快速生成结构化的周报，只需提供关键信息，AI 帮你润色成专业文档。

## 🛠️ 用到的工具
- ChatGPT（免费版即可）
- 可选 Claude（用于二次润色）

## ⏱️ 耗时 & 难度
- 预计时间：10分钟
- 难度：⭐ 新手友好

## 🎯 最终效果
一份结构清晰、语言专业的周报，包含：
- 本周完成工作
- 关键数据指标
- 下周计划
- 需要协调的事项

## 📝 操作步骤

### Step 1：准备关键信息

在记事本或直接在 ChatGPT 对话框里，列出本周关键信息：

```
本周完成：
- 完成用户注册流程重构，转化率提升 15%
- 修复 3 个 P0 bug（登录失败、支付异常、数据丢失）
- 完成 Q2 技术评审文档

下周计划：
- 开始支付模块重构
- 完成性能优化方案
```

### Step 2：输入 Prompt

将以下 Prompt 复制到 ChatGPT：

```
你是一位资深技术主管，请帮我写一份周报。

【本周完成】
- 完成用户注册流程重构，转化率提升 15%
- 修复 3 个 P0 bug（登录失败、支付异常、数据丢失）
- 完成 Q2 技术评审文档

【下周计划】
- 开始支付模块重构
- 完成性能优化方案

要求：
1. 语言专业、简洁
2. 突出成果和数据
3. 结构清晰（本周完成 / 数据指标 / 下周计划 / 需要协调）
4. 500字以内
```

**ChatGPT 会生成类似这样的周报：**

> **本周完成工作**
>
> 1. **用户注册流程重构**：优化注册路径，转化率提升 15%，达到 42%
> 2. **P0 Bug 修复**：解决登录失败、支付异常、数据丢失问题，系统稳定性提升
> 3. **Q2 技术评审**：完成文档编写，明确 Q2 技术方向和资源分配
>
> **下周工作计划**
>
> 1. 启动支付模块重构，预计 2 周完成
> 2. 完成性能优化方案，目标降低 API 响应时间 30%

### Step 3：调整优化

如果 ChatGPT 的回复不够符合你的风格，可以继续对话调整：

- "语气再正式一点"
- "加上具体的技术细节"
- "缩短到 300 字"

## ⚠️ 注意事项 / 避坑

1. **不要直接粘贴公司敏感信息**（项目名称、内部系统名称等），用泛称代替
2. **一定要人工审核**，ChatGPT 可能生成不准确的信息
3. **数据要真实**，AI 生成的数据看起来很真，但可能是编的
4. **根据公司文化调整语气**，有些公司偏好简洁，有些偏好详细
