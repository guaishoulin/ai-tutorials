# 关于本站

## 🎯 这个网站是做什么的？

**AI 实战教程**是一个开源的、手把手教你用 AI 解决实际问题的教程站。

我们不谈空洞的概念，只讲**能立刻上手**的实战技巧。

---

## 👨💻 作者是谁？

资深程序开发者，20年深耕程序各个方面。

这个网站的内容都是**自己用过、验证过**的技巧，不是 AI 生成的泛泛而谈。

---

## 🤝 如何贡献？

这个网站是开源的！如果你有好的 AI 使用技巧，欢迎贡献教程：

1. Fork 这个仓库
2. 在 `docs/tutorials/` 下添加你的教程（Markdown 格式）
3. 提交 Pull Request

教程格式参考已有文章即可。
